Frequently asked Questions
  • 1. What are meta tags in HTML?

    Meta tags provide metadata about the HTML document. They are used for things like SEO, defining character sets, and viewport settings.

  • 2. State some basic design elements?

    Common design elements include color, line, shape, space, texture, typography, and balance.

  • 3. What is load balancing?

    Load balancing is a technique to distribute incoming network traffic across multiple servers to ensure no single server becomes overwhelmed.

  • 4. What does npm stand for?

    NPM stands for Node Package Manager. It is used to install, update, and manage packages in Node.js projects.

  • 5. What is scope in JavaScript?

    Scope refers to the current context of code, which determines the accessibility of variables. JavaScript has global, function, and block scopes.

  • 6. How are JavaScript and jQuery different?

    JavaScript is a programming language, while jQuery is a library that simplifies JavaScript operations, especially DOM manipulation and event handling.

  • 7. What is the difference between HTTP and HTTPS?

    HTTPS is the secure version of HTTP. The key difference is that HTTPS uses SSL/TLS encryption to secure data in transit, while HTTP sends data in plain text.

  • 8. What are CSS preprocessors?

    CSS preprocessors like Sass, Less, and Stylus extend CSS with features like variables, nesting, mixins, and functions, which are compiled into regular CSS.

  • 9. What is responsive web design?

    Responsive web design is an approach where websites adapt to different screen sizes and devices using flexible grids, layouts, images, and CSS media queries.

  • 10. What is the DOM?

    The Document Object Model (DOM) is a programming interface for HTML and XML documents that represents the page structure as nodes and objects that can be manipulated with JavaScript.

  • 11. What is AJAX?

    AJAX (Asynchronous JavaScript and XML) is a technique for creating fast, dynamic web pages that update content asynchronously without reloading the entire page.

  • 12. What is a REST API?

    REST (Representational State Transfer) is an architectural style for designing networked applications using HTTP methods (GET, POST, PUT, DELETE) to access and manipulate resources.

  • 13. What is version control?

    Version control is a system that records changes to files over time, allowing you to recall specific versions later. Git is the most widely used version control system.

  • 14. What is the difference between SQL and NoSQL databases?

    SQL databases are relational with structured schemas (MySQL, PostgreSQL), while NoSQL databases are non-relational with flexible schemas (MongoDB, Cassandra), better suited for unstructured data.

  • 15. What is Docker?

    Docker is a platform for developing, shipping, and running applications in containers - lightweight, standalone packages that include everything needed to run an application.
