FAQ
  • 1. What are meta tags in HTML?

    Meta tags provide metadata about the HTML document. They are used for things like SEO, defining character sets, and viewport settings.

  • 2. State some basic design elements?

    Common design elements include color, line, shape, space, texture, typography, and balance.

  • 3. What is load balancing?

    Load balancing is a technique to distribute incoming network traffic across multiple servers to ensure no single server becomes overwhelmed.

  • 4. What does npm stand for?

    NPM stands for Node Package Manager. It is used to install, update, and manage packages in Node.js projects.

  • 5. What is scope in JavaScript?

    Scope refers to the current context of code, which determines the accessibility of variables. JavaScript has global, function, and block scopes.

  • 6. How are JavaScript and jQuery different?

    JavaScript is a programming language, while jQuery is a library that simplifies JavaScript operations, especially DOM manipulation and event handling.

  • 7. What is Content Security Policy?

    CSP is a security feature that helps prevent cross-site scripting (XSS), clickjacking, and other code injection attacks by defining trusted content sources.
